# Bài viết đầu tiên

Xin chào mọi người.

Đây là blog cá nhân của mình, nơi mình chia sẻ về lập trình và công nghệ.

## Nội dung blog

Mình sẽ viết về:

- ASP.NET Core
- React
- Electron
- SQL Server
- Windows
- GitHub

## Mục tiêu

Mục tiêu của blog là ghi lại kiến thức, kinh nghiệm thực tế và các lỗi mình từng gặp khi làm dự án.
