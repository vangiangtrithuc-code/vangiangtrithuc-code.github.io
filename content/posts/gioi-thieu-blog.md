# Giang Blog là gì?

Đây là blog cá nhân mình dùng để ghi lại kiến thức lập trình, kinh nghiệm làm dự án và các lỗi thực tế trong quá trình làm việc.

## Nội dung chính

- ASP.NET Core
- React
- Electron
- SQL Server
- GitHub
- Windows

## Mục tiêu

Mục tiêu của blog là chia sẻ kiến thức thực tế, dễ hiểu và có thể áp dụng ngay.
