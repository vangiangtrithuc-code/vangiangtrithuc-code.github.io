# Tạo blog miễn phí bằng Hugo và GitHub Pages

Hugo là công cụ tạo website tĩnh rất nhanh, phù hợp để làm blog cá nhân hoặc blog kỹ thuật.

## Vì sao chọn Hugo?

- Miễn phí
- Tốc độ nhanh
- Không cần database
- Dễ deploy lên GitHub Pages
- Viết bài bằng Markdown
